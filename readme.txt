Slideshow CK is a responsive slideshow plugin that shows your images with nice effects.

== Description ==

Slideshow CK is an image slider that can display your images or videos with nice effects. It is compatible with mobiles, responsive design (adaptive width) and you can slide with your fingers on your mobile. You can easily manage the slides with a Drag & Drop interface.

<h3>Features</h3>

* Unlimited slides
* Ergonomic admin interface with Drag & Drop
* Can display images or videos
* Multiple options to congifure the slideshow
* Mobiles compatible, you can slide with your fingers
* Responsive design (adapts the image to the width of the container)
* You can add a link on any slide
* Captions with HTML allowed (to put some links for example)
* Display order normal or shuffle/random
* Show the slider using a widget
* Show the slider anywhere using the shortcode (into your article/post for example)
* Load the slider anywhere using a PHP code (into your theme, or where you want)
* Automatic load of images from a folder (Pro version)
* Automatic load of images from Flickr (Pro version)
* Automatic load of images from Posts (Pro version)
* Automatic load of images from WooCommerce products (Pro version)
* Automatic load of images from Google Photos (Pro version)
* Option to show/hide caption
* Option to show/hide title
* Option to show/hide description
* Option to limit number of slides
* Option to limit text length
* Option to give link position (full slide, caption, title, button, none)
* Option to render a button for the link, with custom text and css class
* Open your links in a Lightbox (Pro version)

<p>Get more informations and see the demo on https://www.ceikay.com</p>

== Installation ==

1. Unzip the package `slideshow-ck.zip`
2. Upload the folder `slideshow-ck` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
